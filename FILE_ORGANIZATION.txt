DEPENDENCY ORDER (Load in this order):
======================================
1. Config/ - Base configurations
2. Systems/Character/ - Character setup
3. Systems/Inventory/ - Inventory setup
4. Systems/Progression/ - Progression setup
5. Systems/Combat/ - Combat setup
6. Systems/Enemy/ - Enemy setup
7. Systems/World/ - World setup
8. Systems/Loot/ - Loot systems
9. Systems/UI/ - UI systems
10. Utilities/ - Final utilities

KEY SYSTEMS & THEIR PURPOSE:
=============================
Character Systems - Handle player appearance and initialization
Combat Systems - Manage attacks, damage, and combat mechanics
Inventory Systems - Manage items, equipment, and inventory UI
Enemy Systems - Manage enemy AI, animations, and behavior
Progression Systems - Handle leveling, experience, and stats
Loot Systems - Handle drops, pickups, and bonuses
UI Systems - Display information to players
Config Systems - Store all configuration data
